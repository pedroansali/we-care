O arquivo a seguir está dividido em 4 pastas principais:

- assets: Onde estão todas as imagens utilizadas no projeto, como logos, ícones e planos de fundo.

- javascript: Onde está o tratamento da exibição dos elementos HTML, mais precisamente no que diz respeito à página de Cadastro.

- pages: Lá estão localizadas as páginas HTML de login, cadastro, tela inicial e perfil.

- styles: Esta pasta está dividida em outras 3 subpastas. Nela contém todos os módulos, funções, componentes, variáveis, dentre outros itens que compõem a estilização das páginas HTML.

    - abstract: Nesta subpastas, estão localizados 3 arquivos:
        - funcions: Que são funções criadas a fim de reduzir a redundância aumentando a reutilização de blocos de códigos através dessas funções
        - reset: Esta folha de estilização está responsável por retirar qualquer estilização que venha por padrão nos navegadores a fim de garantir 100% de personalização para o desenvolvedor
        - variables: Nesta folha, estão contidas as variáveis que serão utilizadas em todas as outras folhas SCSS, como variáveis para o estilo da fonte, paleta de cores, etc.

    - components: Nesta subpasta estão as folhas de estilização padrão para o conteúdo principal, cabeçalho e rodapé.

    - style-pages: Nesta subpasta estão as estilizações individuais para cada página HTML, como a tela de login, a tela de cadastro, tela principal e tela de perfil.
